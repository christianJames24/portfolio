My Portfolio

This project is my portfolio.
I've changed how the portfolio looks. If you want to see what it looked like before,
Click here: Old Portfolio


The first thing I've done was change the structure for the portfolio. The old one was very hard to work with and I didn't know where to go with it, so I just changed it all.
I also wanted to give the text more space, so main is wider now.

The next thing I did was change the color scheme. I don't even like green that much, so I just everything to a blue monochromatic color scheme including the icons.
There is also a new footer, which was supposed to look like a beach.
I made everything have a blue shadow that looks like a glow because it looked very boring before I did that. It looks like blue crystals glowing, which is cool.

There is a new portfolio button on the top left of the website, that takes you to a new page.

The wireframes don't need to be changed in my opinion, because the pages haven't really changed enough for me to redo them. There is a new wireframe for the new page though.


LATE ADDITION: I added an order form of the first volume of Lol Comics. It is on the comics page. (It doesn't actually work.)
